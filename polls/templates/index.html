How to make this App for beginners .

Let's start with Daigo application steps.

1      cd /Users/naomichimasumoto/Documents/canape/src  今回はDcuments保存

2      django-admin startproject (プロジェクト名)

3      cd src

%     python3 manage.py runserver


コマンドすると基本的な、adminのウェブができました。

同じ名前の特殊なフォルダと、manage.pyファイル（管理用）が自動的に作られます。

注意すべきは同じ名前の特殊なフォルダの中に、全体を設定するsettings.py と

URLの設定ファイルurls.pyがある。まずはこの二つに作業加えるので重要です。

プロジェクト始めよう-ここまでの詳しい説明

次に自分で表示するWebページを作る

このプロジェクトにmyappという名のアプリを作ります。


python3 manage.py startappmyapp


全体設定ファイルつまり main/settings.py を開いてINSTALLED_APPS に”'myapp',”をセットする
DEBUG = True開発中はこのままで公開する前に" False "にする。

”myapp”の中にtemplatesフォルダを新規に作りindex.htmlファイル、about.htmlファイル、base.htmlファイルを作る。

”myapp”下templatesフォルダの中myappに作り中にmyapp_list.htmlファイル他の関連htmlファイルを増やしていく。

”myapp”の中にstaticフォルダを新規に作りcssフォルダ内index.cssファイルを作る。

”プロジェクト名”の直下にstatic_localフォルダを新規に作りcssフォルダを作る中にheader.cssなどファイルを増やす。

“myapp”の中 views.py  にhtmlを返しViewを追加する。

“myapp”の中 なければ作ったurls.py にパスなど　作ったViewとURLを結びつける。

全体設定ファイルのurls.pyにも観に行くよう(include)に書き加える。
